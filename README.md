## Descrição do Projeto

Este projeto é parte do TDE-1 da disciplina de Experiência Criativa. Ele consiste em um sistema full-stack utilizando **Node.js** no backend e **React** no frontend, com banco de dados **MySQL** para armazenamento de informações. O sistema permite gerenciar jogos, incluindo adição, listagem e manipulação de dados.

---

## Tecnologias Utilizadas

- **Frontend:** React, Vite  
- **Backend:** Node.js, Express, Cors, MySQL2, Nodemon  
- **Banco de Dados:** MySQL  
- **Outras ferramentas:** Visual Studio Code, Git
